# AirBnB-Stream-Data-Ingestion
Build a simulated data pipeline for Airbnb booking data that integrates various AWS services, demonstrating real-time data processing, filtering, and storage.

## Technologies Used
  1. AWS SQS
  2. AWS Lambda
  3. AWS S3
  4. AWS Event Bridge
  5. Python
  6. Git
